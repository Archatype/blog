Applications:

Atom

When it comes to text editors, Atom is
my all-time favorite. It looks nice, everything
is customizable, and it has many options to
give the user a comfortable experience.

Sketch

Photoshop is an amazing application, but
I like sketch a lot more. Its less demanding,
more simple, cheaper, and has everything
I need.

Prepros

I dont use prepros to its fullest extent
because I dont use preprocessors (yet...),
but I love to use prepros to live preview my
websites.

Github App

I dont use the Github app as much as the
terminal, but its nice to have if I dont want
to use the terminal.

Terminal

Although the Github app is easy to use, the
terminal is a lot more flexible.

Google Chrome

Chrome is my all-time favorite browser. I use
all browsers for cross-browser testing, but use
chrome as my main browser.

Organization:

Trello

Trello is great for me because it helps me focus on one thing at a time.

Dropbox

Dropbox is my go-to for storage. 1TB of cloud storage for
not too big of a cost.

Google Calender

I started using Google Calender because I noticed that my schedule & time management skills were terrible.

Learning:

Youtube

If I want to learn something new, Youtube would be the first site I would go to.

HTML & CSS and Javascript & Jquery book
By: Jon Duckett

These two books helped me incredibly when starting out in
web development. I am still reading the Javascript one, and
I still look through the HTML&CSS book when I want a refresher
on a certain concept. Check out the book here (An affiliate link).

Treehouse

Treehouse helped me out a bit, especially with Javascript.
The problem was that I already dove into programming
before treehouse, so it was really boring to go through
all the stuff I lready knew to complete a stack.

Mozilla Developer Network (MDN)

This site has everything you need to know about web dev
(Well.. HTML, CSS, and Javascript). Need to know what a
function does? MDN. Need a web API refrence? MDN.

Code School

I havent used codeschool yet, but have been told that its
targeted towards intermediate programmers, so I thought
that I should add it to my list for future use.

Utilities:

Google Fonts

Not much to say but GFonts is the bes place to get fonts.

Placehold.it

This thing is amazing. It eliminates the stress factor of
having to manually resize images constantly to get a right
fit on your website.

Animate.css

Great tool if you dont want to make your own animations. Dont reinvent the wheel.

Pixlr

This site is great for quick editing. Its no photoshop, but its nice to have.

Foundation

I may use foundation in the future, but id like to get the
hang of writing code by scratch before getting into frameworks.
